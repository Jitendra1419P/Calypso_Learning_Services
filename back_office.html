Introduction to Calypso Back Office

October 26 -30, 2020 Virtual Classroom

The course consists of several modules, each module is designed to have hands-on exercises. The courses are instructed by Calypso Learning Services’ dedicated staff and members of our Subject Matter Experts’ team. Calypso Bank in a Box environments are provided to each participant for demonstrations, configurations, and learning exercises.

The training will be a virtual instructor led training and each attendee will be provided with access to unique instance of cloud-based Calypso environments.

Each participant will receive access to Calypso Learning Service eLearning portal to access training material, slides, and course exercises.

For registration or enrollment details, please email : learning@calypso.com


COURSE DATE:
October 26 -30, 2020


COURSE FEE:
USD 2,500 (excl. taxes)


COURSE LOCATION:
Virtual Classroom


COURSE LEVEL:
Foundation

PREREQUISITES:
We recommend that you understand Calypso objects, data model, and workflows

TOPICS INCLUDE

Module 1: Introduction to Calypso

  • Calypso Architecture
  • System Components, Engines and Servers

Module 2: Static Data Setup

  • Legal Entities, Contacts, Accounts
  • Settlement Instructions
  • Trading Books
  • Hands-On Exercises: Create PO, CP and define SDI

Module 3: Calypso Workflows

  • Trade Workflow & Lifecycle Actions
  • Transfer Workflow & Settlements
  • Message Workflow
  • Generate Messages
  • Hands-On Exercises: Book Trades & BO Objects (Xfer, Messages) & Apply Lifecycle Actions

Module 4: Calypso Task Station

  • Operations Dashboard
  • Task Priorities, Task Internal Reference
  • Hands-On Exercises: Build Task Station Report

Module 5: Fees & Scheduled Tasks

  • Real-Time / Batch Processing
  • Fee Definition, Fee Grid, Fee Config
  • Hands-On Exercises: Define Automatic & Manual Fees; Run Batch

Module 6: Access Permissions

  • Define User Groups, Users Defaults
  • Book Access, Workflow Access
  • Data Authorization

Module 7: Reporting Framework

  • Report Launcher & Report Generation
  • Trade, Transfer, Message Report
  • Portfolio Manager
  • Cash and Security Inventory Report
  • Hands-On Exercises: Define Reports and Save Templates
